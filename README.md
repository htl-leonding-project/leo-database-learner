## How to create gh-pages

* Copy the `.env.template`-file to `.env`
* fill in your git-username and git-email

```
GIT_GLOBAL_MAIL=<your email>
GIT_GLOBAL_USER_NAME=<your git - username>
```

* run the script `run-inside-docker.sh`
